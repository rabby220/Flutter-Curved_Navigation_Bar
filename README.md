# curve_navigation_drawer

<p> Add dependency: </p>

```
dependencies:
  curved_navigation_bar: ^1.0.3 #latest version

```

<p> Easy to use: </p>

```
home: Scaffold(
        bottomNavigationBar: CurvedNavigationBar(
            items: const [
              Icon(Icons.home),
              Icon(Icons.favorite_border),
              Icon(Icons.search),
              Icon(Icons.settings),
              Icon(Icons.share),
            ]),

```
